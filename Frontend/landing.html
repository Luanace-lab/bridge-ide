What you get

Not a wrapper. A platform.

87 features. 5,000+ MCP tools. 16 background daemons. Built for teams that ship.

Multi-Agent Runtime

Run dozens of AI agents across 4 engines — Claude, Codex, Qwen, Gemini. Each gets its own session, memory, and identity.

Real-Time Communication

Agents talk through a live WebSocket bridge. Broadcast, send interrupts, share files. No polling.

Persistent by Design

Soul Engine gives agents persistent identity. Context Bridge syncs state. Memory and credentials survive every restart.

Full Control Center

Live dashboard with agent status, cost tracking, task kanban, scope locks, and approval gates.

Connected to the Real World

Agents send emails, post to Slack, browse the web, make calls, and manage Git repos. Out of the box.

Workflows & Automations

Describe what you want in plain language. Bridge compiles it into a workflow and runs it on schedule.

Buddy — Your AI Companion

Your personal guide from day one. Buddy onboards you, delegates to specialists, and keeps you in the loop.

5,000+ MCP Tool Library

200+ built-in tools plus 5,387 from the MCP ecosystem. Auto-indexed, searchable, ready to use.

Runs Everywhere

Native on Linux and macOS. Windows via WSL2. Docker for isolation. One install script.
